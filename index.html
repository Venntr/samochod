Konfiguracja — jeden raz
Dane przechowywane są w pliku data.json w Twoim repozytorium GitHub — działają na każdym urządzeniu. Wykonaj poniższe kroki, a potem wpisz swoje dane w sekcji CONFIG w kodzie strony.
1
Utwórz repozytorium na GitHubMoże być prywatne. Wgraj do niego dwa pliki: index.html (ta strona) i data.json (plik z danymi). Włącz GitHub Pages: Settings → Pages → Deploy from branch → main.
2
Utwórz Fine-grained Personal Access TokenGitHub → Settings → Developer settings → Fine-grained tokens → Generate new token. Wybierz tylko to repozytorium. Uprawnienie: Contents: Read and write. Resztę zostaw bez zmian.
3
Uzupełnij sekcję CONFIG w kodzie HTMLW pliku index.html znajdź blok z komentarzem /* ── CONFIG ── */ i wpisz swój login GitHub, nazwę repozytorium oraz wygenerowany token.
⚠️ Token PAT będzie widoczny w kodzie HTML. Korzystaj z prywatnego repozytorium i ogranicz zakres tokenu wyłącznie do tego repo.
